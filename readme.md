# Three.js Journey

## Setup
Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

``` bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```

## Project Overview
This project is a playable ad of a game named **Cube Sort** in which we are designing a voxel. This game was originally made in Unity, and the goal is to convert the existing Unity (C#) scripts to JavaScript and render it using Three.js.
